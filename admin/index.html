Google Sheets API Connection

📋 How to get your Apps Script URL:
  1. Open your Google Sheet → Extensions → Apps Script
  2. Paste the google-apps-script.gs file contents
  3. Run SETUP_EVERYTHING once (creates all sheets + seeds products)
  4. Click Deploy → New Deployment
  5. Type: Web App
  6. Execute as: Me
  7. Who has access: Anyone ← very important
  8. Click Deploy → Authorize → Copy the URL
  9. Paste the URL below and click Save, then Test
URL must end with /exec
⚠️ If you see "Failed to fetch":
The most common cause is the deployment setting "Who has access" is not set to Anyone. Go back to Apps Script → Deploy → Manage deployments → Edit → change access → re-deploy → copy the new URL.

API Security Token

This token authenticates admin operations to your Google Apps Script backend.
It is pre-configured automatically. Run SETUP_SECURITY() in Apps Script once to activate server-side protection.

After rotating: run SETUP_SECURITY() in Apps Script again with the new token.

🚀 First-Time Setup Checklist

Complete these steps once after deploying your Google Apps Script.

  1. Open your Google Sheet → Extensions → Apps Script
  2. Paste the entire google-apps-script.gs file contents
  3. Run SETUP_EVERYTHING() → creates all sheets & seeds data
  4. Deploy → New Deployment → Web App → Execute as Me → Anyone → Deploy
  5. Copy the Web App URL → paste below in Settings → API URL field
  6. SETUP_SECURITY() runs automatically — tokens are pre-configured
